Wyświetl sortowalną listę wszystkich reklam umieszczonych na Twoim blogu za pośrednictwem platformy reklamowej SeoPilot.pl. 



== Description ==

Używasz SeoPilot.pl? Zobacz listę wszystkich reklam wyświetlanych na Twoim blogu wprost w panelu administracjnym WordPressa. 

* Automatycznie wykrywa kod serwisu
* Możliwość przenoszenia definicji CSS do sekcji <head>
* Widget z kodem reklam
* Możliwość automatycznego umieszczania kodu reklam w stopce
* Wyświetla wszystkie reklamy wyświetlane w danym serwisie w formie sortowalnej tabeli
* Zlicza i wyświetla liczbę wszystkich reklam wyświetlonych w danym serwisie, liczbę unikalnych adresów docelowych oraz unikalnych reklamodawców 
* Osobno zlicza i wyświetla reklamy o treściach erotycznych

Nie używasz jeszcze SeoPilot.pl? Zarejestruj się już teraz: http://bit.ly/seopilotpl 



EN: 

This plugin is dedicated to the Polish advertising platform SeoPilot.pl. It lists all the ads that are placed on the site in form of a sortable table.


== Installation ==

1. Rozpakuj archiwum i wgrał cały katalog `spdwp` do katalogu z wtyczkami (zazwyczaj `/wp-content/plugins/`)
2. Aktywuj plugin w panelu WordPress-a

== Frequently Asked Questions ==

= Czy plugin wymaga podania hasła do systemu SeoPilot.pl? =

Nie. Plugin w żaden sposób nie łączy się systemem SeoPilot.pl, tylko analizuje plik z listą reklam, który znajduje się na Twoim serwerze.
